# Getting Started with Your Wiki

Welcome! This folder is your personal knowledge base for this project.

Claude Code can search everything here using the `/wiki` skill.

## How to add notes

Create any `.md` file in this folder. Name it whatever makes sense — Claude will find it by filename and content.

Examples:
- `decisions.md` — why you made key choices
- `architecture.md` — how the system fits together
- `notes-2024-01.md` — freeform session notes

## How to search

In Claude Code, type:

```
/wiki                    # list everything available
/wiki authentication     # find notes about auth
/wiki why did we         # search for decision rationale
```

Claude reads the most relevant file and answers your question directly.

## Tips

- You don't need perfect organization. Claude searches by content, not just filenames.
- Short focused files work better than one giant doc.
- Write notes right after a decision or discovery, while it's fresh.
